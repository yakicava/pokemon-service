# Pokemon Battle Playground

ポケモンの **図鑑情報・タイプ相性・簡易対戦・SV準拠バトル** を扱う  
Spring Boot 学習用の Web アプリケーションです。

本アプリは「いきなり複雑な対戦を作る」のではなく、  
**図鑑 → タイプ相性 → 疑似対戦 → SV準拠バトル**  
という段階的な拡張を前提に設計しています。

---

## 主な機能

### 図鑑機能（アプリの基盤）
- 図鑑番号を入力してポケモン情報を取得
- 名前 / タイプ / 画像を表示
- 外部API（PokeAPI）を利用

### タイプ相性 疑似対戦
- 攻撃側タイプと防御側タイプから相性を判定
- 数値計算を伴わないシンプルな対戦ロジック
- バトルロジック分離の学習を目的とした機能

### SVバトル（オート対戦）
- 図鑑番号を2体入力してバトル実行
- Lv50固定
- 技はSVでLv50までに覚える技から自動選択
- 先手判定・ダメージ計算・ターン進行を実施
- 勝者・ターン数・最終HP・ターンログを表示

---

## 設計方針

- Controller と Service の責務を明確に分離
- ドメインロジック（タイプ相性・バトル計算）は Service 層に集約
- 例外は Service で発生させ、Controller で画面制御
- 小さく作って段階的に拡張できる構成を重視
